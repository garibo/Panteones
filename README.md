Panteones App
================
Sistema para registro de difuntos.
Esta aplicacion fue construida utilizando los frameworks:
- [AngularJS](https://angularjs.org/)
- [Codeigniter](https://codeigniter.com/)
- [Chart.js](http://www.chartjs.org/)
